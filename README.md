# Završni rad 
## Detekcija jezgri stanica uz pomoć algoritama evolucijskog računanja
Opisati problem identifikacije jezgre stanice u medicinskoj analizi. Istražiti uobičajene postupke rješavanja i navesti  
postojeća rješenja. Ostvariti programski sustav za analizu slike i identifikaciju područja stanične jezgre. U programskom  rješenju koristiti postupke **evolucijskog računanja** za optimizaciju parametara transformacije slikovnih elemenata. U sustav  dodati mogućnost prikaza rada algoritma i konačnog rješenja. Ispitati učinkovitost ostvarenog sustava na zadanim ispitnim  primjerima. Radu priložiti izvorne tekstove programa, dobivene rezultate uz potrebna objašnjenja i korištenu literaturu.
